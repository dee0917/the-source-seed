# Flux - Stunning Landing Page

A premium landing page template designed with a focus on minimalism, typography, and fluid motion.

## Tech Stack
- **React** (UI Logic)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Advanced Physics-based Animations)
- **Lucide React** (Iconography)
- **Vite** (Build Tool)

## Key Design Features
- **Deep Dark Aesthetic**: Pure `#030303` background for infinite depth.
- **Glassmorphism**: Subtle backdrops and borders for a layered, physical feel.
- **Micro-interactions**: Spring-based physics on buttons and cards.
- **Film Grain Overlay**: A subtle CSS-animated noise texture to break the digital perfection.
- **Fluid Typography**: Inter and JetBrains Mono combination for a high-tech, editorial look.

## Project Structure
- `src/App.jsx`: Main landing page composition.
- `src/components/ui/`: Reusable, atomic design components.
- `src/index.css`: Custom CSS utilities and global styles.
- `tailwind.config.js`: Custom color palette and animation definitions.

## Installation
```bash
npm install
npm run dev
```
