# iFreeTime - Redev Task Instructions for Little C

## Project Overview
iFreeTime: A universal time-sovereignty PWA. Reclaim time by allowing users to share "Free slots" with groups without revealing calendar details.

## Tech Stack (MANDATORY)
- **Frontend**: Next.js 15 (App Router), Tailwind CSS 4, Shadcn UI, Framer Motion.
- **Backend**: Python 3.12 (FastAPI), Pydantic AI.
- **PWA**: PWA-ready (manifest, icons, standalone mode).
- **Style**: Premium, mobile-first, high-end iOS app feel (Dark mode base).
