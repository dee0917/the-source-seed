# Prompt Optimizer (最佳提示詞優化器)

這項技能使代理人能夠將用戶的指令轉化為最高效、最精確的 AI 提示詞。

## 優化模型 (The Golden Prompt Architecture)

當 D 下達指令時，自動套用以下框架進行重構：

1.  **Role (角色)**: 定義執行該任務的最頂尖專家身份。
2.  **Context (上下文)**: 注入當前系統狀態、已安裝技能與歷史背景。
3.  **Objective (任務目標)**: 用極其精確的動詞描述最終產出。
4.  **Constraints (約束條件)**: 包含安全規則 (sw-security) 與格式限制。
5.  **Chain-of-Thought (思考鏈)**: 誘導模型逐步思考以確保邏輯無誤。

## 執行流程

1.  **攔截**: 接收原始輸入。
2.  **蒸餾**: 提取關鍵意圖。
3.  **重組**: 根據上述框架生成「最佳提示詞」。
4.  **自我反思**: 檢查該提示詞是否可能引起歧義或安全風險。

## 使用範例

*   **D 的輸入**: "幫我寫一個備份腳本"
*   **優化產出**:
    > "作為一名 Linux 資深系統管理員，請為這台運行 Ubuntu 24.04 的 VPS 撰寫一個 Bash 備份腳本。目標是將 /root/.openclaw/workspace 同步至一個指定的遠端路徑。腳本必須包含：1. 錯誤處理邏輯；2. 日誌紀錄功能；3. 基於時間戳的壓縮備份。請確保不包含任何硬編碼的敏感憑證。"
